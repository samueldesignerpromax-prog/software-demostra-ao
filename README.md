# SalesBot Pro - Plataforma de Vendas com IA

## 🚀 Demonstração Profissional para Portfólio

Sistema completo de vendas e atendimento com chatbot inteligente, dashboard analítico e captura de leads.

### ✨ Funcionalidades

- **Dashboard Profissional** com métricas em tempo real
- **Chatbot Inteligente** com respostas automáticas
- **Catálogo de Produtos** com simulação de compra
- **Captura de Leads** com API integrada
- **Modo Demonstração** com notificações simuladas
- **Design Responsivo** para todos os dispositivos
- **Animações Modernas** para melhor experiência

### 🛠️ Tecnologias Utilizadas

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animações)
- Recharts (gráficos)
- Lucide React (ícones)

### 📦 Instalação Local

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/salesbot-pro.git

# Entre na pasta
cd salesbot-pro

# Instale as dependências
npm install

# Execute em desenvolvimento
npm run dev

# Build para produção
npm run build
npm start
